# Transparent Keyboard

This is a custom mechanical keyboard with a fully clear acrylic case and transparent keycaps.  
The goal of this project was to build a simple keyboard that shows all of the internal parts, lighting, and wiring.

## About

This keyboard is designed to be easy to see through and easy to understand.  
Every major part is visible, including the PCB, LEDs, switches, and wiring.  
It uses QMK firmware and a Teensy microcontroller.

---

## Firmware

This keyboard uses QMK firmware.

**Flashing steps**

1. Download QMK Toolbox.
2. Plug the Teensy 2.0 into your computer.
3. Press the boot button on the Teensy.
4. Flash the provided `.hex` firmware file using QMK Toolbox.
